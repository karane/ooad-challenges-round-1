# Challenge 02 — Logger Router System

A flexible logging library that routes log entries to multiple backends (Console, File, ELK/HTTP) through a single API. Each appender can be configured with a minimum log level and run synchronously or asynchronously.

## How to Run

**Prerequisites:** Java 25 (via SDKMAN or system install), Maven 3.x.

### Run the demo

```sh
mvn package exec:java
```

### Run the tests

```sh
mvn test
```
